# Реализация
Сделал Канбан доску с тремя колонками(Todo, In Progress, Done). Добавление задачи при нажатии на кнопку "Добавить задачу" - открывается модалка с формой. Фильтрация задач с помощью интерактивных табов: Все, Без описания, Обьемные, также поиск по задачам с задержкой 200ms. У каждой задачи есть 3 функции: удалить, изменить, переместить. При удалиении задача удаляется из состояния. При нажатии на изменить открывается форма, аналогочная при создании, с кнопками "отмена" и "обновить". при нажатии на иконку перемещения открывается режим перемежения , перетаскиваем задачу в другую колонку.
Также при нажатии на каждую задачу открывается режим просмотра - модальное окно с данными об задаче, где нельзя изменять данные. 
При нажатии на "Tab" происходит фокус на поле input , также с помощью "Tab" происходит обход всех элементов страницы. Нпри наведении на карточкуи нажатии "Space/Enter" происходит октрытие режима редактировния. При заполнении формы, нажимая "Enter" происходит выход из формы и сохранение резултата, также при нажатии "Escape" форма закрывается
Реализован механизм undo/redo для отмены последней задачи или пвтора последней задачи. Происходит сохраниение последнего действия в localStorage
# Архитектура
* components/store.js - начальные данные берутся из seed.json и сохраняются в localStorage, затем происходит взаимодецствие с данными только через localStorage: загрузка, обновление данных
* components/router.js - создает маршруты /board и /task/:id, устанавливает начальный путь, отслеживает изменения URL и парсит путь. Действия: /board - закрывает модалку, показывает доску 
/task/:id - открывает модалку с задачей, подсвечивает карточку. 
* components/action/dragDrop.js - запоминает перетаскиваемую задачу, добавляет CSS-класс. Перетаскивание над колонкой - разрешает дроп, показывает индикатор перемещения, определяет новую колонку и статус. Добавляет действие в историю Undo (старый и новый статус)
* components/action/search.js - Поиск: Реактивный поиск - срабатывает при вводе с задержкой 200мс. Подсветка результатов - выделяет найденный текст. Фильтрация: 3 фильтра: "Все", "Без описания", "Объемные" (>50 символов)
* components/ui/modal.js - Открытие/закрытие модалок - создание и редактирование задач. Управление фокусом - ловушка Tab/Shift+Tab, возврат фокуса при закрытии. Обработка клавиатуры - Enter для сохранения, Escape для закрытия. Разные режимы - создание (save-btn) vs редактирование (update-btn).
Доступность - правильные ARIA-атрибуты, управление inert. Интеграция с роутером - навигация при открытии/закрытии
* components/undo/undoManager.js - Хранит историю действий - создание, редактирование, перемещение, удаление задач. Сохранение в localStorage - история переживает перезагрузку страницы. Ограничение размера - максимум 50 действий, автоматическая очистка старых. Добавление действия - сохраняет данные + обратные данные для отката. Undo/Redo - применяет обратные/прямые действия, обновляет UI. Синхронизация - автоматическое сохранение состояния после каждого действия
* components/utils/utils.js - Поддерживает 3 тега: **жирный**, _курсив_, `код`. Защита от XSS: экранирование HTML-символов перед обработкой. Санитизация: удаляет неразрешенные теги, оставляя только текст. Двойная защита: экранирование - преобразует <, >, & в HTML-entities, фильтрация - удаляет любые теги кроме разрешенных (strong, em, code)
* components/view/columns.js - Загрузка данных - асинхронно получает задачи из хранилища
Применение фильтров - поиск + фильтрация по типу. Очистка колонок - удаляет предыдущее содержимое. Группировка по статусу - распределяет задачи по колонкам (ToDo/In Progress/Done). Рендеринг: создание карточек - для каждой задачи в колонке, пустые состояния - показывает сообщение "Нет задач"
* components/view/createTask.js - Генерация ID - на основе максимального существующего ID + 1. Добавление в историю Undo - перед сохранением данных. Сохранение и перерисовка - обновляет хранилище и интерфейс
Создание карточки: доступный интерфейс - tabIndex, ARIA-атрибуты, роль button, обработка кликов - навигация по карточке (исключая клики по кнопкам), подсветка поиска - выделяет найденный текст в заголовке и описании
